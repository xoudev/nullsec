# NULLSEC

Personal portfolio. [nullsec.fr](https://nullsec.fr)

---

## Stack

- **Next.js 16** — App Router, Turbopack
- **GSAP** — animations and scroll effects
- **Lenis** — smooth scroll
- **Tailwind CSS v4**
- **TypeScript**

## Design system

Four colours, no gradients.

| Token | Value | Role |
|---|---|---|
| `--color-void` | `#0A0A0B` | background |
| `--color-bone` | `#F2EFE8` | foreground |
| `--color-blood` | `#E63946` | accent |
| `--color-ash` | `#707070` | metadata |

Fonts: `Instrument Serif` italic (display) · `Inter` (body) · `JetBrains Mono` (UI)

## Development

```bash
npm install
npm run dev
```
